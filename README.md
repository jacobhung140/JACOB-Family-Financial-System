# JACOB 家庭財務系統（Family Financial System）

這是一套由 Jacob 建立、並以 GitHub + AI 為核心的「家庭財務管理系統」。  
目標是讓家庭財務變得 **制度化、可追蹤、可傳承、可自動化**，並能夠在未來 10～30 年持續運作。

---

# 🎯 系統目的（Purpose）

1. 建立一套可重複、可傳承的家庭財務管理流程  
2. 每年進行一次完整的家庭財務健康檢查  
3. 透過 19 項量化指標，評估家庭財務的強弱  
4. 建立年度資料庫（data/）作為長期紀錄  
5. 透過 AI 協助自動生成年度報告與改善計畫  
6. 最終達成「自動化家庭財務系統」

# 📊 19 項財務健康指標（Annual 19 Metrics）

這套系統以 19 個財務指標為核心，涵蓋：

- 安全性（緊急預備金、流動性、負債風險…）
- 投資（長期投資、集中風險…）
- 人的風險（收入、職涯、健康…）
- 長期風險（長照、法律、稅務…）
- 家庭治理與紀律

每項以 1～3 分量化，並形成年度雷達圖。

---

# 📅 年度流程（Annual Workflow）

1. 啟動年度檢查  
2. AI 問 19 題  
3. AI 自動生成：  
   - 分數總表  
   - 年度總結  
   - 改善計畫  
   - 雷達圖資料  
4. 存入 `data/YYYY/`  
5. 家庭會議討論  
6. 更新年度行動計畫  
7. 完成年度紀錄

# 🤖 自動化（Automation Roadmap）

本系統設計為可逐步自動化：

### Phase 1：制度化（已完成）
- 流程、規則、模板、資料夾結構

### Phase 2：半自動化（進行中）
- AI 自動生成年度資料（2026 已完成）

### Phase 3：完全自動化（未來）
- GitHub Actions 自動建立年度資料夾  
- AI 自動寫入所有檔案  
- 自動 commit  
- 自動更新 Dashboard（GitHub Pages）  
- AI 每年自動提醒進行年度檢查
